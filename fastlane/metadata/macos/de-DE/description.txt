Nft Folder ist ein persönliches, privates Tool zum Offline-Betrachten von hochauflösender Kunst.

Es ist kostenlos, quelloffen und ablenkungsfrei.

Durchsuchen Sie über 700 ausgewählte Kollektionen. Genießen Sie den Vollbildmodus, Bild-in-Bild oder stellen Sie einen Bildschirmschoner ein.

Verwenden Sie es auch, um alle NFTs für jede Ethereum-Adresse herunterzuladen.
