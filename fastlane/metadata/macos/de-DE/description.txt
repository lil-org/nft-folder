Nft Folder ist ein persönliches, privates Tool zum Offline-Betrachten von hochauflösender Kunst. Es ist kostenlos, quelloffen und ablenkungsfrei.

Durchsuchen Sie über 700 ausgewählte Sammlungen. Vollbild, Bild-in-Bild oder Bildschirmschoner.

Sie können es auch verwenden, um alle NFTs für eine beliebige Ethereum-Adresse herunterzuladen.
