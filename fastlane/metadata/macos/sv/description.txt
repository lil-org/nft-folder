Nft Folder är ett personligt, privat verktyg för att se konst i hög upplösning offline. Det är gratis, öppen källkod och fritt från distraktioner.

Bläddra bland över 700 utvalda samlingar. Spela upp i helskärm, bild-i-bild eller sätt som skärmsläckare.

Du kan också använda det för att ladda ner alla nft för valfri Ethereum-adress.
