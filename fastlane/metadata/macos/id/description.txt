Nft Folder adalah alat pribadi untuk melihat seni beresolusi tinggi secara offline. Gratis, open-source, dan bebas gangguan.

Telusuri lebih dari 700 koleksi unggulan. Putar dalam mode layar penuh, gambar-dalam-gambar, atau atur sebagai screensaver.

Anda juga dapat menggunakannya untuk mengunduh semua NFT dari alamat Ethereum mana pun.
