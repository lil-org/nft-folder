Nft Folder adalah alat pribadi dan privat untuk melihat seni beresolusi tinggi secara offline.

Gratis, open-source, dan bebas dari gangguan.

Jelajahi lebih dari 700 koleksi unggulan. Nikmati dalam mode layar penuh, picture-in-picture, atau jadikan sebagai screensaver.

Anda juga dapat menggunakannya untuk mengunduh semua nft dari alamat Ethereum mana pun.
